# Customer Feedback Automation Workflow (n8n)

Workflow tự động xử lý và phân loại phản hồi khách hàng.

## Tính năng chính
- Trigger khi có form submission
- Sử dụng Google Gemini AI để tự động phân loại phản hồi (Tốt / Trung bình / Cần xử lý)
- Tự động lưu dữ liệu có cấu trúc vào Google Sheets
- Xử lý real-time

## Công cụ sử dụng
- n8n
- Google Gemini AI
- Google Sheets
